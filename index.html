sh.exe: hexo: command not found

By Wu Bo

发表于 2015-06-15

今天在git命令行工具中使用hexo的时候，总是提示我找不到hexo命令，如：
sh.exe”: hexo: command not found
但是我运行 npm install -g hexo-cli安装成功，且没有错误提示。
想起之前系统新建了一个用户，可能windows下的环境变量不对，在windows下的path中加入如下路径，问题解决：
C:\Users\Username\AppData\Roaming\npm\
