分享几个日常用的chrome插件

发表于 2015-06-26   |   分类于 技术文章   |  
  1. AdBlock
    AdBlock是chrome浏览器中安装量高达4000万,几乎能够屏蔽整个互联网上的广告的一款非常著名的广告屏蔽插件。
  2. All Downloader Professional
  3. Bootstrap Twitter Offline Docs
    Bootstrap离线文档
  4. Easy Auto Refresh

    Easy Auto Refresh可以简单的设置网页的自动刷新时间。
  5. feedly
    RSS订阅
  6. Image Downloader
    批量下载所有的图片。您也可以通过宽度、高度、URL过滤网页图片。
  7. Infinity新标签页
    新标签页应用，重绘了几百个图标，支持自定义搜索，集成天气，带有备忘与 Todo。
  8. JavaScript Errors Notifier
    在Chrome地址栏中以图标形式提示JavaScript错误
  9. JSONView
    JSONView使JSON格式的数据有类似XML的折叠展开模式，方便检查局部数据，使JSON数据整齐格式化。
  10. LastPass
    在线密码管理器，支持多种平台之间的数据同步。
  11. OneTab
    把所有标签页收缩到一个窗口来减少内存占用
  12. Page Ruler
    精确地测量出当前页面某个元素或者是整个页面的大小尺寸。
  13. Proxy SwitchyOmega
    配合GoAgent翻墙使用
  14. Resolution Test
    测试网页在不同分辨率下的显示效果，可以查看当前测试分辨率。
  15. Tabs Outliner
    浏览器旁边新建一个窗口，把所有的标签页按照域名归类放好。
  16. Visual Event
    快速查看 DOM 上绑定的 JS 事件
  17. Wappalyzer
    查看网站使用了哪些框架和技术
  18. Web Developer
    在javascript控制台的基础进行了扩展，方便html和css的调试
  19. WEB前端助手(FeHelper)
    FE助手功能：前端编码规范检测、页面性能检测、栅格检测、时间戳转换工具、正则表达式工具、字符串编解码、代码美化工具、JSON格式化、二维码生成、Javascript执行效率检查等。
  20. WhatFont
    查看网页字体
  21. 网页截图 - FireShot
    网页截图插件
